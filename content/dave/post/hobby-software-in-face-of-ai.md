In the office many corporations are mandating use of AI for software development, but for side projects or personal projects, how much AI makes sense? If I'm writing software on my own time for fun as a hobby, I can use AI to maximize my productivity and make better use of my limited free time, but then am I still having fun writing software and do I still have a hobby? 

Existential questions time: why am I doing this in the first place? I've had to write down the aspects of working on prersonal projects that makes me want to do this in the first place:

* Using the resulting project (or creating something ridiculous only for the sake of it)
* Helping others with the resulting project
* Solving high level design problems
* Solving low level coding problems
* Making the code beautiful
* Learning new things
* Practicing existing skills

I find myself starting different projects for different combinations of these reasons. 

The first two really just care about the result and it can be hard to sustain and finish projects if that's the only reason I'm working on them. AI can help with these cases.
